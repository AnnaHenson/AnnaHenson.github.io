Problem Solving

What was the problem?

The problem was:

  • Each property has a value assigned to a new object.

I created a variable with an empty object and defined gradebook, but then forgot a property should be assigned to the value of a new object.
So, the property Elizabeth was not being assigned to an empty object, which meant the property value was not recognised in the code. The computer could not read the property of Elizabeth so was shown as undefined, which came up as an error on the console on the right.

What problem solving techniques did you use?

My first option was to try some pseudocode code and write down the steps in English to utilise a reasoning tool, but it did not go to plan. The other technique was to try and read the error messages in relation to errors for code failure, but my final throw of the dice was to ask both a developer and facilitator what the hell was wrong.
