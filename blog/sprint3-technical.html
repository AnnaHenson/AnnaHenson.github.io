What is the DOM? How might you use it?

DOM stands for document object model and is a hierarchal tree structure, which gives a structural representation of HTML and uses HTML elements upon which the Dom operates.

In essence the DOM represents a web page which can be manipulated once the web page is opened in the browser or window. The HTML gives the hierarchical structure a path for the DOM to interact with other coding languages such as HTML, CSS and JavaScript. So, the analogy is a tree in my garden with roots as the window and a trunk which acts as the document. The trunk allows us to climb up and access more content in the top of the tree by having more branches with more nodes (elements).
